Курсовой проект "Калькулятор для путешественника"

Веб-приложение "Калькулятор для путешественника" предназначено для конвертации валют, измерений длины и подсчёта итоговой суммы бюджета. Целевая аудитория: гиды, организаторы поездок, туристы.
